API for IP Information and Geolocation

With this API you will be able to get information about an IP address such as ISP, Geolocation, and much more. Our outputs are all either in json or plaintext. We also provide you with an audio file for the IP!

Note: each audio file gets deleted after 100s

curl https://[domain]/api/v1?ip=8.8.8.8&data=all
